A certificação **AWS Certified AI Practitioner (AIF-C01)** possui:

* 65 questões
* 90 minutos
* Nota mínima de aprovação: 700/1000
* Nível Fundamental ([Amazon Web Services, Inc.][1])

Os domínios cobrados são:

* Fundamentos de IA e Machine Learning (20%)
* Fundamentos de IA Generativa (24%)
* Aplicações de Foundation Models (28%)
* IA Responsável (14%)
* Segurança, Compliance e Governança (14%) ([Documentação AWS][2])

### Semana 2 — Serviços AWS

# Armadilhas comuns

Não decore respostas. Entenda os conceitos.

Saiba diferenciar:

* IA x ML x Deep Learning
* Fine-tuning x RAG
* Embedding x Prompt
* Token x Context Window
* Hallucination x Bias
* Bedrock x SageMaker
* Foundation Model x Modelo treinado pelo cliente

Essas comparações aparecem com frequência em questões baseadas em cenários. ([Documentação AWS][2])

## O que eu faria no seu lugar

Como você já possui experiência em AWS e uma certificação anterior, eu faria um plano de **21 dias** focado em eficiência:

* **Dias 1–7:** conceitos de IA + IA generativa.
* **Dias 8–14:** todos os serviços AWS de IA e ML.
* **Dias 15–21:** simulados intensivos, revisando cada erro até entender o motivo da resposta correta.

Esse método aproveita sua base de cloud e direciona o esforço para o que realmente muda em relação ao Cloud Practitioner.

Também posso montar um **bootcamp completo**, semelhante ao que fizemos para a certificação da Red Hat, contendo:

* cronograma diário;
* resumos de cada domínio;
* mapas mentais;
* mais de **500 questões comentadas** no estilo da prova;
* simulados progressivos até você atingir um nível de aprovação consistente na primeira tentativa.
